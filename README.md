# Shadowrocket Configuration

Этот репозиторий содержит конфигурационный файл для приложения Shadowrocket, который помогает оптимизировать маршрутизацию интернет-трафика, блокировать нежелательные сервисы и обеспечивать безопасные и эффективные подключения.

## Особенности
- **Поддержка сервисов Apple:** Обеспечение стабильной работы App Store, iCloud, Apple Music.
- **Блокировка рекламы и трекеров:** Использование встроенных фильтров от AdGuard и StevenBlack.
- **Обход блокировок:** Пользовательские правила для безопасного доступа к заблокированным ресурсам.
- **Оптимизация потоковых и игровых сервисов:** Маршрутизация трафика Netflix, YouTube, Twitch и других через прокси.

## Важное предупреждение
Я использую оператора связи **Теле2**, и данный конфигурационный файл оптимизирован для работы с его сетью. У других операторов мобильной связи или провайдеров интернета стабильность и производительность могут отличаться. Пожалуйста, тестируйте на своей стороне и редактируйте файл под свои нужды. Данный репозиторий создан лично для себя, но возможно вам что-то отсюда поможет

## Использование
1. Скачайте файл `shadowrocket.conf`.
2. Импортируйте файл в приложение Shadowrocket.
3. При необходимости отредактируйте правила под свои нужды.
